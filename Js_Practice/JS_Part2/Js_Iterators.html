.forEach()

The .forEach() method is a way to execute a function on each item in an array. It’s often used when you need to perform actions on each element, but don't need to create a new array or return a value.

Syntax: array.forEach(callback(currentValue [, index [, array]]) [, thisArg]);

  • callback: This is the function that is called for every item in the array.
  • currentValue: The value of the current element in the array.
  • index (optional): The index of the current element (position in the array).
  • array (optional): The original array that .forEach() was called upon.
  • thisArg (optional): The value to use as this when executing the callback function.

  • Basic Example

    const numbers = [1, 2, 3, 4, 5];
    numbers.forEach(function(number) {
    console.log(number);
    });
    //Output: 1 2 3 4 5

  • Using Index: index of each element inside the .forEach() function:

    const fruits = ['apple', 'banana', 'cherry'];
    fruits.forEach(function(fruit, index) {
    console.log(`Index ${index}: ${fruit}`);
    });
    Output// Index 0: apple Index 1: banana Index 2: cherry

  • Using thisArg:(refer to original array)You can provide a value for thisArg if you want to use a specific context inside your callback.
    names refer to the original array.

    const names = ['Gurvinder', 'Singh', 'kathy'];
    names.forEach(function(element,index,array) {
    console.log(element,index,array);
    });
    Output//
    Gurvinder 0 [ 'Gurvinder', 'Singh', 'kathy' ]
    Singh 1 [ 'Gurvinder', 'Singh', 'kathy' ]
    kathy 2 [ 'Gurvinder', 'Singh', 'kathy' ]
    names.forEach(function(array) {
    console.log(names);
    });
    output//
    Gurvinder
    Singh
    kathy

  • Using Arrow Functions

    const colors = ['red', 'green', 'blue'];
    colors.forEach(color => {
    console.log(color);
    });

The .map() Method

map() function, which is a method used with arrays in JavaScript .map() is called on an array, it takes an argument of a callback function and returns a new array! .

.map() works in a similar manner to .forEach()— the major difference is that .map() returns a new array.

Syntax array.map((element, index, array) => { });

  • element: The current element being processed in the array.
  • index: The index of the current element being processed.
  • array: The original array that map() was called upon.

  • Basic Example:Transforming an array of numbers to their squares:

    let numbers = [1, 2, 3, 4, 5];
    let squares = numbers.map(num => num * num);
    console.log(squares); // Output: [1, 4, 9, 16, 25]

  • Using Index: Appending the index to each element:

    let names = ['gurvinder', 'singh', 'GS'];
    let namesWithIndex = names.map((name, index) => `${name} #${index}`);
    console.log(namesWithIndex); // Output: ["gurvinder #0", "singh #1", "GS #2"]

  • Converting Objects: Extracting a specific property from an array of objects:

    let users = [
    { name: 'gurvinder', age: 25 },
    { name: 'singh', age: 30 },
    { name: 'dharuv', age: 35 }
    ];
    let names = users.map(user => user.name);
    console.log(names); // Output: ["gurvinder", "singh", "dharuv"]

  • Chaining Methods: Combining map() with other array methods:

    let numbers = [1, 2, 3, 4, 5];
    let result = numbers
    .map(num => num * 2) // Double each number
    .filter(num => num > 5); // Keep only numbers greater than 5
    console.log(result); // Output: [6, 8, 10]

The .filter() Method

The .filter() method in JavaScript is used to create a new array with all elements that pass a test implemented by a provided function.

. The callback function for the .filter() method should return true or false depending on the element that is passed to it. The elements that cause the callback function to return true are added to the new array.

Syntax: array.filter((element, index, array) => { });

  • element: The current element being processed in the array.
  • index: The index of the current element being processed.
  • array: The original array that .filter() was called upon.

  • Basic Filtering: Filtering out odd numbers from an array:

    let numbers = [1, 2, 3, 4, 5, 6];
    let evenNumbers = numbers.filter(num => num % 2 === 0);
    console.log(evenNumbers); // Output: [2, 4, 6]

  • Filter users whose age is greater than 30

    let users = [
    { name: 'Gurvinder', age: 25 },
    { name: 'singh', age: 30 },
    { name: 'sham', age: 35 }
    ];
    const userAgeAbove30 = users.filter(element => element.age > 30);
    console.log(userAgeAbove30);
    // Output: [{ name: 'sham', age: 35 }]

Understanding .map() vs. .filter()

.map(): Transforms each element in the array based on a function and returns a new array of the same length as the original. The function should return a new value for each element.

.filter(): Creates a new array containing only the elements that pass a test. The function should return a boolean value (true or false), and only elements that return true are included in the new array.

map() vs filter(): Use .map() when you need to transform data, and .filter() when you need to select a subset of elements based on a condition.

.find(): Use when you want the first matching element.
.filter(): Use when you want all matching elements

  • return a value of each array element for new array

    let ageChecks = users.map(element => element.age > 30);
    console.log(ageChecks);
    // Output: [false, false, true]

  • Return and add the value into new array where the condition is true.

    let userAgeAbove30 = users.filter(element => element.age > 30);
    console.log(userAgeAbove30);
    // Output: [{ name: 'Charlie', age: 35 }]

findIndex()

Calling .findIndex() on an array will return the index of the first element that evaluates to true in the callback function.

Syntax: array.findIndex((element, index, array) => {
// Return true if the condition is met{
});

  • Finding the index of the first number greater than 20:

    let numbers = [10, 20, 30, 40, 50];
    let index = numbers.findIndex(num => num > 20);
    console.log(index); // Output: 2 (30 is the third element in the array)

reduce()

The .reduce() method in JavaScript is a powerful array method used to accumulate or reduce the values of an array into a single result based on a provided function. It’s useful for operations like summing values, concatenating elements, or combining arrays.

Syntax: array.reduce((accumulator, currentValue, currentIndex, array) => {
// Return the updated accumulator
}, initialValue);

  • accumulator: The accumulated value returned from the last iteration. On the first iteration, it's the initialValue if provided, or the first element of the array if no initialValue is provided.
  • currentValue: The current element being processed in the array.
  • array: The original array that .reduce() was called upon (optional).
  • initialValue: A value to use as the first argument to the first call of the callback function. If no initialValue is provided, the first element of the array will be used as the initial accumulator value.

  • Summing Array Elements

    let numbers = [1, 2, 3, 4, 5];
    // Sum all numbers in the array
    let sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
    console.log(sum); // Output: 15

  • Counting Occurrences

    let fruits = ['apple', 'banana', 'apple', 'orange', 'banana', 'banana'];
    let counts = fruits.reduce((accumulator, fruit) => {
    accumulator[fruit] = (accumulator[fruit] || 0) + 1;
    return accumulator;
    }, {});

    Explanation : accumulator[fruit] refers to the count for the current fruit. If accumulator['apple'] exists, it increments the count by 1; otherwise, it sets the initial value to 0 if accumulator[fruit] is undefined.
