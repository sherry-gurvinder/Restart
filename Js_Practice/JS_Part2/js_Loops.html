For Loop

A loop is a programming tool that repeats a set of instructions until a specified condition, called a stopping condition is reached.

  • The For Loop: for (initialization; condition; increment) { // Code to be executed in each iteration }

    for (let i = 1; i <= 5; i++) { console.log(i); }

  • Nested Loops: Nested loops in JavaScript are loops placed inside other loops. They are useful when you need to perform operations on multi-dimensional structures like matrices or tables.

    Basic Syntax for (let i = 0; i < outerLimit; i++) { for (let j = 0; j < innerLimit; j++) { // Code to be executed } }

The While Loop

a while loop repeatedly executes a block of code as long as a specified condition evaluates to true. It's particularly useful when you don't know in advance how many times you'll need to iterate.

  • while (condition) { // Code to be executed while the condition is true }

    let count = 1; while (count <= 5) { console.log(count); count++; // Increment the count to eventually break the loop }

Do...While Statements

you want a piece of code to run at least once and then loop based on a specific condition after its initial run. This is where the do...while statement comes in.

  • do { // Code to be executed } whil

    let count = 1; do { console.log(count); count++; // Increment the count } while (count <= 5);

The break Keyword

The break keyword allows programs to “break” out of the loop from within the loop’s block.

  • hello will only print 3 times

    for (let i = 0; i < 99; i++) { if (i > 2 ) { break; } console.log('hello.'); }

for...of

The for...of loop is useful for iterating over arrays, strings, and other iterable objects in a more straightforward way than using a traditional for loop

  • Using for...of loop to iterate through the array

    const rooms = ['#308', '#302', '#303']; for (const room of rooms) { (room); }

for...in

for...in loop is used to iterate over the enumerable properties of an object. It allows you to loop through all the keys of an object

  • Basic Syntax

    for (const key in object) { if (object.hasOwnProperty(key)) { // Code to execute for each property console.log(`${key}: ${object[key]}`); } }

  • This loop iterates over the keys (or property names) of the object.

    const person = { name: 'Alice', age: 30, occupation: 'Engineer' }; for (const key in person) { if (person.hasOwnProperty(key)) { // Optional, to avoid iterating over inherited properties console.log(key + ': ' + person[key]); } }
