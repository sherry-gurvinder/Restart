Creating a Grid

To turn an HTML element into a grid container, you must set the element’s display property to one of two values:

  1. grid — for a block-level grid.

    {display:grid;}

  2. inline-grid — for an inline grid.

    {Display-inline-grid;}

By default, grids contain only one column. If you were to start adding items, each item would be put on a new row;

Creating Rows: By default, the rows are sized to fit evenly inside the grid.

To specify the number and size of the rows, we are going to use the property grid-template-rows.

This property is almost identical to grid-template-columns. Take a look at the code below to see both properties in action.

.grid { display: grid; width: 1000px; height: 500px; grid-template-columns: 100px 200px; grid-template-rows: 10% 20% 600px; }

Fraction:CSS Grid introduced a new relative sizing unit — fr, like fraction.

You may already be familiar with several types of responsive units such as percentages (%), ems and rems.

Using fr makes it easier to prevent grid items from overflowing the boundaries of the grid.

.grid { display: grid; border: 1px solid red; width: 1000px; grid-template: 2fr 3fr / 2fr 3fr 4fr ; }

In this example, the grid will have three rows and three columns. The rows are splitting up the available 400 pixels of height into four parts. The first row gets two of those parts, the second row gets one, and the third row gets one. Therefore the first row is 200 pixels tall, and the second and third rows are 100 pixels tall.

It is possible to use fr with other units as well. When this happens, each fr represents a fraction of the available space.

.grid { display: grid; width: 100px; grid-template-columns: 1fr 60px 1fr; }

Repeat: repeat()

grid-template-columns: repeat(3, 100px);

grid-template-rows: repeat(2,50px)

Repeat is particularly useful with fr. For example, repeat(5, 1fr) would split your table into five equal rows or columns.

Second parameter of repeat can have multiple values

grid-template-columns: repeat(5,200px 100px)

This code will create 5 columns where the first, third and fifth columns will be 200 pixels wide and the second and fourth will be 100 pixels wide.

minmax: its a function to restrict the dimension of the row or column, minmax(minimum, maximum) minmax(100px , 500px)

.grid { display: grid; grid-template: 1fr/ repeat(2, 350px minmax(100px, 150px)); }

the first, 3rd and fifth columns will always be 350 pixels wide, no matter the size of the grid. The second and 4th column, however, will vary in size as the overall grid resizes. The second column will always be between 100 and 150 pixels wide.
