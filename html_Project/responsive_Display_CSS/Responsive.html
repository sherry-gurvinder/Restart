FLEX:Any element can be a flex container. Flex containers are helpful tools for creating websites that respond to changes in screen sizes. Child elements of flex containers will change size and location in response to the size and position of their parent container.

Display:flex

For an element to become a flex container, its display property must be set to flex. div.container { display: flex; }

All divs with the class container are flex containers. If they have children, the children are flex items. it will change the behavior of its child elements. Child elements will not begin on new lines.

Flex Grow
display:flex = All the items of flex container align to each other in single line and if the width of the flex container shrink or expand. All flex items will be shrink and expand equally

The flex-grow property allows us to specify if items should grow to fill a container and also which items should grow proportionally more or less than others.

Below is the example if only set the flex-grow property under Left Div to

#left { flex-grow: 1; }

then it will take all the width of the flex container and Center Div, right div element shrink to their min-width property i.e
